# 📱 GitHub Pages 网页部署教程（无需 Git）

## 🎯 总耗时：10 分钟

---

## 步骤1：创建 GitHub Personal Access Token（3分钟）

### 1.1 访问 Token 创建页面
打开浏览器访问：https://github.com/settings/tokens/new

### 1.2 填写信息
- **Note (备注名称)**：`Deploy Asset Calculator`
- **Expiration (过期时间)**：选择 `No expiration`（永不过期）
- **Select scopes (选择权限)**：
  - ✅ 勾选 `repo`（完整的仓库访问权限）
  - 其他都不用勾选

### 1.3 生成 Token
1. 滚动到页面底部
2. 点击绿色按钮 **Generate token**
3. **立即复制**生成的 token（格式类似：`ghp_xxxxxxxxxxxx`）
4. ⚠️ **重要**：这个 token 只显示一次，务必保存好！

---

## 步骤2：创建 GitHub 仓库（2分钟）

### 2.1 创建新仓库
1. 访问：https://github.com/new
2. 填写信息：
   - **Repository name (仓库名)**：`asset-calculator`
   - **Description (描述)**：资产配置计算器
   - ✅ 选择 **Public**（公开）
   - ✅ 勾选 **Add a README file**
3. 点击底部绿色按钮 **Create repository**

### 2.2 记下你的仓库地址
格式：`https://github.com/你的用户名/asset-calculator`

---

## 步骤3：上传项目文件（3分钟）

### 3.1 进入上传页面
1. 在你刚创建的仓库页面
2. 点击 **Add file** → **Upload files**

### 3.2 上传文件
**方法A：拖拽上传**
1. 打开文件夹 `d:\pytools`
2. 选择以下文件（按住 Ctrl 多选）：
   ```
   ✅ index.html
   ✅ style.css
   ✅ calculator.js
   ✅ admin.html
   ✅ admin.js
   ✅ schemes-admin.html
   ✅ schemes-admin.js
   ✅ README.md
   ✅ qrcode.html
   ```
3. 拖拽到浏览器的上传区域

**方法B：点击上传**
1. 点击 **choose your files**
2. 选择上述文件
3. 点击"打开"

### 3.3 提交上传
1. 在页面底部 **Commit changes** 区域
2. 保持默认信息：`Add files via upload`
3. 点击绿色按钮 **Commit changes**
4. 等待上传完成（1-2分钟）

---

## 步骤4：启用 GitHub Pages（2分钟）

### 4.1 进入设置
1. 在仓库页面，点击顶部的 **Settings**（设置）
2. 左侧菜单找到 **Pages**

### 4.2 配置 Pages
1. **Source (来源)**：选择 `Deploy from a branch`
2. **Branch (分支)**：
   - 第一个下拉框选择 `main`
   - 第二个下拉框选择 `/ (root)`
3. 点击 **Save** 按钮

### 4.3 等待部署
1. 页面会刷新
2. 顶部会显示一个蓝色框，提示正在部署
3. 等待 1-2 分钟

### 4.4 获取网址
1. 刷新页面（F5）
2. 顶部会显示一个绿色框
3. 里面有你的网站地址：
   ```
   https://你的用户名.github.io/asset-calculator/
   ```
4. 🎉 点击 **Visit site** 查看你的网站！

## 🔄 更新网站内容

### 方法1：网页编辑（适合小改动）
1. 在 GitHub 仓库找到要修改的文件
2. 点击文件名
3. 点击右上角的铅笔图标 ✏️
4. 修改内容
5. 拉到底部点击 **Commit changes**
6. 等待 1-2 分钟，网站自动更新

### 方法2：重新上传（适合大改动）
1. 在 GitHub 仓库点击要替换的文件
2. 点击删除按钮 🗑️
3. 点击 **Add file** → **Upload files**
4. 上传新文件
5. 提交更改

## ❓ 常见问题

### Q: 网站显示 404 Not Found？
**A:** 等待 1-2 分钟，GitHub Pages 需要构建时间。刷新页面重试。

### Q: 如何确认 Pages 是否启用？
**A:** 进入 Settings → Pages，看到绿色提示框就是成功了。

### Q: 可以改仓库名吗？
**A:** 可以！Settings → General → Repository name，改完后网址也会变。

### Q: 如果忘记 Token 了？
**A:** Token 只用于首次上传，网页操作不需要 Token。

### Q: 数据会丢失吗？
**A:** 不会。数据保存在用户浏览器本地，跟 GitHub 无关。
