## 🔑 管理员操作指南

### 1️⃣ 登录系统
1. 打开 https://ched666.github.io/asset-calculator/login.html
2. 点击"管理员"按钮
3. 输入密码：`admin123`
4. 点击"管理员登录"

### 2️⃣ 修改产品配置
1. 登录后进入"后台配置"页面
2. 修改存款产品和理财产品的利率、FTP等参数
3. 点击"保存配置"

### 3️⃣ 管理推荐方案
1. 进入"方案管理"页面
2. 添加、编辑或删除银行推荐方案
3. 点击"保存所有方案"

### 4️⃣ 发布到云端（重要）

#### 首次配置GitHub Token
1. 访问 https://github.com/settings/tokens
2. 点击"Generate new token" → "Generate new token (classic)"
3. 设置Token名称（如：asset-calculator）
4. 勾选权限：
   - ✅ **repo**（完整仓库权限）
5. 点击"Generate token"
6. **复制生成的Token**（只显示一次，请妥善保管）

#### 在管理页面配置
1. 进入"方案管理"页面
2. 找到"☁️ 云端配置发布"区域
3. 填写：
   - GitHub用户名：`ched666`
   - GitHub仓库名：`asset-calculator`
   - GitHub Token：粘贴刚才复制的Token
4. 点击"💾 保存配置"

#### 发布配置
1. 修改完产品配置或推荐方案后
2. 点击"🚀 发布到云端"按钮
3. 等待1-2分钟，系统会自动更新GitHub仓库中的 `default-config.json`
4. 新用户访问时将自动加载最新配置

## 📋 常见问题

### Q: 如何修改管理员密码？
A: 编辑 `login.html` 文件，修改第 96 行的密码：
```javascript
const ADMIN_PASSWORD = 'admin123'; // 改成你想要的密码
```

### Q: 普通用户能修改配置吗？
A: 不能。普通用户只能查看和使用计算器，无法访问管理页面。

### Q: GitHub Token安全吗？
A: Token仅保存在您的浏览器本地（localStorage），不会上传到服务器。但建议定期更新Token。

### Q: 发布到云端失败怎么办？
A: 检查以下几点：
1. GitHub Token是否正确，是否有repo权限
2. 用户名和仓库名是否正确
3. 网络连接是否正常
4. 查看浏览器控制台错误信息

### Q: 如何导入导出配置？
A: 在"方案管理"页面底部有"导出配置"和"导入配置"按钮：
- 导出：下载当前配置为JSON文件
- 导入：从JSON文件恢复配置
